A WhatsApp-integrated tool that turns voice notes into structured meeting documentation. Team leads connect a business number, add their team (no accounts needed for members), and send voice notes or text messages. The system transcribes, filters casual chatter, and returns formatted minutes with attendees, key points, decisions, and action items — delivered back via WhatsApp and archived in a web dashboard. Built with TanStack Start, Bolt Database (row-level security for data isolation), and WhatsApp Cloud API, with admin oversight and full data deletion support for RA 10173 compliance. This web app is still a work in progress, features are under active development and may change.

Stock Balanse is a work-in-progress inventory management system for small to medium businesses. It supports bulk Excel imports, real-time stock tracking, outbound transaction logging (sales, damage, transfers) with a timestamped audit trail, and a live dashboard showing total stock, low-stock warnings, and top deducted items over 30 days. Built with React (TanStack Start), TypeScript, Tailwind, and Supabase (auth, PostgreSQL, row-level security for multi-tenant data isolation). Planned features include granular reporting, user roles, and enhanced exports.
